# COVID-19 Tracker
Use the following Layout Designs (**`Relative Layout, Table Layout, Frame Layout, Absolute Layout`**) and create a small individual project in which there will be 4 or 5 Activities connected with each other by buttons or any other widget of your choice. In each activity use one of the following layout designs to show text or image or anything of your choice. Try to design something meaningful.<br><br>
